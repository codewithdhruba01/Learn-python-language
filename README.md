## üü¢ Beginner Level

### üîπ Arrays & Strings
- [ ] Reverse an array
- [ ] Find the maximum/minimum element
- [ ] Remove duplicates from a list
- [ ] Rotate array by k positions
- [ ] Check if a string is a palindrome
- [ ] Count vowels and consonants
- [ ] Implement Python‚Äôs `split()` manually

### üîπ Basic Math
- [ ] Prime number check
- [ ] GCD and LCM of two numbers
- [ ] Factorial using recursion
- [ ] Fibonacci series
- [ ] Palindromic number check

### üîπ Searching & Sorting
- [ ] Linear search
- [ ] Binary search
- [ ] Bubble sort
- [ ] Selection sort
- [ ] Insertion sort


## üü° Intermediate Level

### üîπ Linked Lists
- [ ] Reverse a linked list
- [ ] Detect a cycle in a linked list
- [ ] Merge two sorted linked lists
- [ ] Find middle of linked list

### üîπ Stack & Queue
- [ ] Implement stack using list or deque
- [ ] Valid parentheses (LeetCode #20)
- [ ] Implement queue using two stacks
- [ ] Next Greater Element (LeetCode #496)

### üîπ Hashing & Dictionary
- [ ] Two Sum problem (LeetCode #1)
- [ ] Group anagrams (LeetCode #49)
- [ ] Longest substring without repeating characters

### üîπ Recursion & Backtracking
- [ ] Generate all subsets of a list
- [ ] Permutations of a list
- [ ] Sudoku solver
- [ ] N-Queens problem


## üî¥ Advanced Level

### üîπ Trees & Binary Trees
- [ ] Inorder, Preorder, Postorder traversal
- [ ] Check if a tree is balanced
- [ ] Lowest common ancestor
- [ ] Serialize and deserialize a tree

### üîπ Heaps & Priority Queues
- [ ] Kth largest element (LeetCode #215)
- [ ] Merge k sorted arrays
- [ ] Top K frequent elements

### üîπ Graphs
- [ ] BFS and DFS traversal
- [ ] Detect cycle in a graph
- [ ] Dijkstra's algorithm

### üîπ Dynamic Programming
